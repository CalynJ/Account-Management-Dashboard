# Account-Management-Dashboard

The primary goal is to provide a clear, step-by-step overview of the current account management review process, focusing on how data is manually pulled, manipulated, and used to generate client and debtor concentration reports.

Key Points:
Assumes the user is already familiar with FactorView reports and navigation.
Focuses on generating reports using data pulled by "Purchase Date" type.
Manual processes involve sorting data, calculating concentration percentages, and generating aging bucket percentages.
Includes guidelines on pulling Client AR Detail and Debtor AR Detail reports, performing manual sorting and calculations, and creating various tables and pivot tables.
The process covers generating insights like client concentration, debtor concentration, and account reviews by representative or client.
Reports and Procedures:
Client Concentration: How to sort client data by highest concentration and calculate aging buckets and percentage concentration.
Debtor Concentration: Similar steps applied to debtor data, focusing on debtor concentrations and aging percentages.
Client Concentration by Account Rep: Steps to extract data specific to each account representative and perform concentration analysis.
Client Account Review: Steps to create aging tables, find top debtors, and manually review ADTP trends.
Additional Manual Insertions:
Client logo, yield report, billing and verification procedures, and action items.
This repository serves as documentation for the process but is not meant for training purposes.
